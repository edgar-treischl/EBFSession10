# EBF: Sitzung 10

Lecture slides for **Empirische Bildungsforschung (EBF)** at FAU Nürnberg-Erlangen, built with [Reveal.js](https://revealjs.com) and [Vite](https://vitejs.dev).

👉 **[View slides](https://edgar-treischl.github.io/EBFSession03/)**

## Development

```bash
make dev      # start dev server at http://localhost:3000
make build    # production build → dist/
make preview  # serve the build locally
```
